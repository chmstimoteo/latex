As próximas atividades que serão desenvolvidas são: 
\begin{enumerate}
\item Desenvolver um Estudo Comparativo entre MLP, SVM, Simulação de Monte Carlo e Regressão Linear para a previsão do impacto dos riscos, utilizando a base de dados PERIL.
\item Desenvolver um Estudo Comparativo entre Redes de Petri, Árvores de Falha, Redes Bayesianas e Lógica Fuzzy para o cálculo do grau de exposição ao risco do projeto, utilizando dados do PERIL.
\item Oferecer um Método de Suporte a Tomada de Decisões Estratégicas de Mitigação de Riscos.
\item Escrever dissertação de mestrado.
\end{enumerate}

A escrita de artigos e da dissertação também será realizada no decorrer das outras atividades.
